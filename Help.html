Help and Documentation


"Even though it is better if the system can be used without documentation, it may be necessary to provide help and documentation. Any such information should be easy to search, focused on the user's task, list concrete steps to be carried out, and not be too large." - Nielsen


The website has several tools such as Search, Maps, Contact Info, etc. However the website does not have a specific help link that is marked, nor is their any particular way to get to the help page. Like most university websites, this website has an IT help desk website. There is a way to contact the webmaster through e-mail, but no direct immediate way to connect to get help immediately. As far as information technologies go there isn't a whole lot the website offers to help you with any technical problems. However as far as the whole site goes there is quite a bit that goes around, such as maps, contact info that allows for the user to get info about the school and its location.
