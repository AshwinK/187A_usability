Help Users Recognize, Diagnose, and Recover from Errors


"Error messages should be expressed in plain language (no codes), precisely indicate the problem, and constructively suggest a solution. " - Nielsen


The error recovery on this site is quite effective, not only does it auto-correct the user, but it provides helpful suggestions to ease the user's experience. This is quite possibly the best feature of this crummy site, and honestly we were quite surprised. When an error occurs with a broken or non-existent URL, the site redirects you to a page giving you multiple choices to make. You can visit the directory, homepage, or search page. This gives the user a lot of freedom, which increases the user's fulfillment of experience.

Also this site has an auto-correct feature built into the search bar. This is quite a relief, and the user is allowed to pick-up as if he had never erred in the first place. The sight has amazing error recovery options. "Error messages should be expressed in plain language (no codes), precisely indicate the problem, and constructively suggest a solution." This website follows the heuristic down to the very letter, it is a well fulfilled quota.
