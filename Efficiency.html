Flexibility and Efficiency of Use


"Accelerators -- unseen by the novice user -- may often speed up the interaction for the expert user such that the system can cater to both inexperienced and experienced users. Allow users to tailor frequent actions." - Nielsen


This site doesn't really feature any accelerators besides the one that can be accessed from the Search Function. For example, we searched the word "Students" and the results on the left showed up. The link we'd been to already was darkened to show we'd visited it already. If I was a consistently-visiting user, then I'd be able to search in the search page for a page I'd already been to during that session and I'd find it pretty quickly. Keep in mind that there are no breadcrumbs and navigation isn't that great, so you're better off just going back to the main site and finding your way back. New users and old users differ only in the fact that older users may "recall" certain navigation paths, whereas new users are left confused and must get to know the site from experience.
